# 📚 Study OS

A simple, beautiful study management app for students and exam aspirants. Plan your tasks, track your focus, manage notes, and watch your progress with real-time analytics.

**[🚀 Try Live Demo](https://snehavish595.github.io/study-os/)**

---

## ✨ What Can You Do?

📋 **Plan Tasks** - Break down your study goals into manageable tasks with deadlines  
⏱️ **Track Focus** - Monitor your study sessions and see how much time you spend on each subject  
📊 **See Your Progress** - Visual charts showing your learning patterns and performance  
📝 **Take Notes** - Write and organize notes directly in the app  
🎯 **Stay Motivated** - Get insights about your study habits and celebrate wins  

---

## 🚀 Quick Start

1. **Visit the app** - Open [the live demo](https://snehavish595.github.io/study-os/) in your browser
2. **Add subjects** - Create subjects you're studying
3. **Create tasks** - Break down what you need to study
4. **Start a focus session** - Use the timer to track study time
5. **Watch analytics** - See your progress update in real-time

## 💻 For Developers

Want to use it locally?

```bash
git clone https://github.com/snehavish595/study-os.git
cd study-os
# Open index.html in your browser
```

**Tech Stack:** Pure JavaScript (77.6%) + HTML (22.4%), no dependencies needed
